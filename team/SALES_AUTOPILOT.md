# SALES_AUTOPILOT (24x7)

## 目标
- 持续优化产品与销售漏斗
- 以订单与回款为核心
- 满足收入/token成本 >= 10

## 永续循环（每次轮询执行）
1. 检查收入看板与线索漏斗
2. 若线索不足：生成/发布新内容并触达
3. 若演示不足：推进预约与跟进
4. 若报价未成交：执行异议处理与限时促单
5. 若交付阻塞：优先修复演示/交付路径
6. 记录日志：daily + iteration + retro

## 触发阈值与动作
- 私信率 < 0.8%：更换选题和CTA
- 预约率 < 15%：重写私信脚本和预约钩子
- 成交率 < 15%：重做报价锚点与案例证明
- token成本异常：启用低token降级策略

## 每日必交付
- 1条获客内容
- 15+触达
- 2场演示目标推进
- 看板更新与复盘
