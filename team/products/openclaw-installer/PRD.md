# PRD - OpenClaw 一键部署包（中国版）

## 目标
为不会安装 OpenClaw 的个人与小团队提供“可落地、可运维、可付费”的一键部署方案（macOS + Windows）。

## 商业定位（双引擎-B）
- 引擎A：项目经理复盘助手（现金流）
- 引擎B：OpenClaw 一键部署包（放大客单与服务收入）

## ICP
1. 个人效率用户（不会命令行）
2. 小团队负责人（希望 1 天内落地）
3. AI 服务商（代部署需求）

## MVP 交付（14天）
1. 安装向导（系统检查、网络检查、依赖检查）
2. macOS 安装包（.pkg）
3. Windows 安装包（.msi/.exe）
4. 一键健康检查（诊断报告）
5. 服务绑定入口（购买、续费、工单）

## 收费模型（建议）
- 安装包授权：¥299/设备（一次性）
- 企业快速部署：¥2999/次（远程交付）
- 运维订阅：¥499/月（升级/巡检/故障支持）
- 团队年包：¥5999/年（含部署+运维）

## 成功指标（30天）
- 付费部署 >= 5 单
- 试用到付费转化 >= 15%
- 安装成功率 >= 90%
- 工单首响 < 2小时

## 风险与对策
- 许可证/商用边界：上线前完成合规检查清单
- 网络环境波动：提供国内镜像/离线依赖包
- 系统差异：先支持主流版本，低版本分级支持
