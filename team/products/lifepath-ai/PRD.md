# PRD - 仙人指路（LifePath AI）v1.0 MVP

## 1. 产品概述

### 1.1 一句话定义
一个“懂你、安全、可行动”的AI生涯顾问，帮助迷茫用户把复杂人生问题转化为当下可执行路径。

### 1.2 目标用户
- C端：18-35岁迷茫中的年轻人（职业发展阶段）
- B端（后续）：企业HR与管理者（人才发展与辅导）

### 1.3 核心价值
- 深度个性化：记住用户长期背景，持续优化建议
- 强执行导向：输出“本周最优行动”与复盘闭环
- 隐私可证明：本地优先 + 审计可查 + 零知识保护

---

## 2. V1 范围（MVP）

### 2.1 目标边界（减法设计）
V1 仅聚焦“职业发展路径建议”，不做全生命周期心理/健康/婚恋等泛咨询。

### 2.2 MVP 功能清单
1. 用户输入采集
   - 主动输入：日记/心情/目标
   - 被动输入：日程、邮件（授权后接入）
2. 记忆与推理引擎
   - Mem0记忆管理（本地优先）
   - LLM生成阶段性行动建议
3. 每日行动闭环
   - 早晨：今日建议
   - 晚间：回顾与画像更新
4. 安全与审计
   - 敏感信息检测与脱敏
   - 合规审计日志与隐私报告

### 2.3 非目标（V1不做）
- 企业管理后台
- 复杂协作权限系统
- 全量多端离线同步

---

## 3. 关键流程

### 3.1 Onboarding（首次体验）
1) 注册/登录（手机号或邮箱）
2) 初始化本地数据库与向量索引
3) 首次问题引导：最近最困扰的职业问题
4) 安全拦截通过后写入记忆
5) 输出首版建议与本周行动项

### 3.2 每日推荐闭环
1) 早晨：根据昨日数据+日程生成今日建议
2) 白天：记录执行情况
3) 晚间：复盘并更新画像
4) 记录审计与隐私证明摘要

---

## 4. 技术架构（实现约束）

### 4.1 技术栈
- 前端：React + TypeScript（移动端优先）
- 后端：FastAPI（REST API + JWT）
- 存储：SQLite（本地）+ 向量索引（Mem0）
- 模型：支持本地推理（Qwen/LLaMA）+ 可切云端

### 4.2 安全机制（必须实现）
- OpenClaw合规拦截层：PII检测、脱敏、权限校验
- Zero Knowledge Proof模块（证明“可用不可见”）
- 审计日志：禁止落原文，仅存脱敏哈希与操作元数据
- API返回头要求：`x-privacy-proof`

### 4.3 架构原则
- 本地优先：目标80%+数据处理在端侧完成
- 可审计：每个敏感操作可追溯
- 可降级：离线可运行核心功能

## 5. 商业模型（V1）

- C端免费版：基础建议（获客）
- C端Pro：¥29/月（深度分析、历史回溯）
- 社区会员：¥99/年（脱敏案例学习）
- 企业版（V2）：按席位/年收费

## 6. 验收指标（MVP）

### 6.1 产品指标
- D7留存 > 20%
- 每周至少1次有效建议点击率 > 40%
- 行动项完成率 > 35%

### 6.2 安全指标
- 敏感数据误出率 = 0（P0）
- 审计日志覆盖率 = 100%
- 隐私证明头覆盖率 = 100%

### 6.3 业务指标
- 首月有效激活用户 >= 100
- Pro转化率 >= 5%

## 7. 开发计划（6周）

### Phase 1（Week 1-2）基础架构
- FastAPI项目初始化
- Auth（JWT）
- Mem0本地记忆模块
- 隐私拦截骨架（PII检测/脱敏）

### Phase 2（Week 3-4）核心能力
- 路径建议引擎
- 每日建议与复盘流程
- 前端核心页面（输入、建议、回顾）

### Phase 3（Week 5-6）质量与安全
- 压测与稳定性优化
- 安全审计与漏洞修复
- MVP试运行与数据回收

---

## 8. 风险与对策

1) 隐私泄露风险（高）
- 对策：本地优先、ZKP、强审计、最小权限

2) 模型幻觉与建议不准（中）
- 对策：引入结构化模板、置信度提示、人工反馈闭环

3) 用户留存不足（中）
- 对策：强化“每日可执行建议”与完成反馈奖励机制
